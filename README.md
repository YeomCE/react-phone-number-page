`React`를 사용하여 전화번호부 홈페이지를 제작했습니다.


`React-Redux`를 처음으로 활용하여 작업한 프로젝트입니다. 전화번호부에 필요한 다양한 기능을 넣으면서도 최대한 React-Redux의 활용에 익숙해질 수 있도록 노력했습니다.


반응형 웹으로 제작하여 다양한 환경 속에서도 레이아웃이나 기능 등이 망가지지 않습니다.
